# 🕵️‍♂️ OSINT Contractor Scoring System (AI Powered)

**Автоматизированная B2B-система скоринга и оценки рисков контрагентов с применением генеративного искусственного интеллекта (LLM).**

Проект решает главную боль коммерческих директоров и служб безопасности в строительном (и не только) секторе: долгий ручной сбор и анализ данных о подрядчиках перед допуском к тендеру. Система консолидирует цифровой след компании (суды, долги, финансы) и выдает готовый аналитический бизнес-отчет за 5 секунд.

## 🚀 Бизнес-ценность
* **Снижение финансовых рисков:** ИИ автоматически выявляет "красные флаги" (несоответствие уставного капитала оборотам, опасные арбитражные дела).
* **Экономия времени:** Заменяет 30-40 минут ручного гугления и анализа баз данных на один клик.
* **Стандартизация решений:** Исключает человеческий фактор — нейросеть оценивает всех подрядчиков по единому жесткому алгоритму.

## 🛠 Технологический стек и Архитектура
* **Python 3.11** — ядро системы.
* **OpenRouter API (LLM Balancer)** — динамический балансировщик нейросетей. Система не привязана к одному вендору (No Vendor Lock-in). Запросы маршрутизируются на самые быстрые и доступные модели (Google Gemini, Meta Llama, Mistral) без необходимости использования VPN.
* **REST API & Requests** — легковесная интеграция без перегруженных SDK.
* **Pydantic (Settings)** — строгая и безопасная валидация переменных окружения.

## ⚙️ Как это работает
1. Пользователь вводит ИНН компании.
2. `DataCollector` собирает "сырые" данные из открытых источников (имитация API Контур.Фокус / Руспрофайл).
3. `RiskAnalyzer` формирует сложный системный промпт и отправляет данные через защищенные заголовки в балансировщик ИИ.
4. Система выдает структурированный отчет: Оценка риска (1-10), Красные флаги, Позитивные факторы и Итоговый вердикт.
